# 대한민국 여행가이드 웹사이트

이 저장소는 대한민국 주요 관광지와 맛집을 소개하는 여행가이드 웹사이트입니다.

## 주요 기능

- 지역별 관광명소 소개
- 현지인 추천 맛집 정보
- 인터랙티브 지도
- 반응형 디자인
- 사용자 요청 게시판

## 포함된 지역

- 여수
- 제주도
- 전주

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Google Maps API
- Bootstrap 4

## 사용 방법

GitHub Pages로 호스팅되어 있으며, 다음 URL에서 접속할 수 있습니다:
https://[사용자명].github.io/korea-travel-guide/
